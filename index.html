Экспорт пишет в выбранную папку: name.html (или name.json для Lottie / Raw JSON) + diagnostic.json (structured-диагностика + список слоёв + warnings) + assets/ (копии footage, если были). Detеrministic: одинаковый вход → одинаковый выход.

Выбор формата: CSS+SVG — для баннеров и кросс-браузерной прод-ready анимации; GSAP — если нужен timeline-контроль; Lottie — для интеграции через lottie-web/LottieFiles (учтите: эффекты/маски/gradients не мапятся); Raw JSON — для диффа и debugging.
