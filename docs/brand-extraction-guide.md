## Что извлекать из каждой композиции

Для каждой композиции ниже — **запросы агенту** в панели. Копируй запрос, вставляй в чат.

---

### Шаг 1. Общая структура

```
Дай подробный summary этой композиции: все слои, их типы, parenting, эффекты, 3D-статус, timing (in/out points)
```

---

### Шаг 2. Цвета и стили

```
Для каждого слоя в композиции: прочитай fill color (если shape или solid), text document properties (font, size, color, justification), opacity. Выведи все значения
```

---

### Шаг 3. Keyframes — Transform

```
Для каждого слоя прочитай все keyframes на Transform (Position, Scale, Rotation, Opacity, Anchor Point). Для каждого keyframe укажи: time, value, easing type, influence in/out
```

---

### Шаг 4. Keyframes — Masks

```
Прочитай маски на всех слоях: mode, feather, opacity, expansion. Если есть keyframes на mask properties — выведи их с timing и значениями
```

---

### Шаг 5. Keyframes — Effects

```
Для каждого эффекта на каждом слое: имя эффекта, matchName, все свойства с текущими значениями. Если есть keyframes на свойствах эффектов — выведи timing и значения
```

---

### Шаг 6. Expressions

```
Прочитай все expressions на всех слоях (get_expression для каждого свойства где есть expression). Выведи полный текст каждого expression
```

---

### Шаг 7. Shape paths (только для логошотов)

```
Для всех shape layers: прочитай shape content — vertices, in/out tangents каждого path. Это нужно для воспроизведения логотипа
```

## Порядок работы

### Блок A: Логошоты (проект Логошоты — edit/Логошот.aep)

> Используй версию «edit» — она новее (1.6M vs 1.3M)

**Открыть:** `Логошоты — edit/Логошот.aep`

| # | Композиция | Шаги | Что получаем |
|---|-----------|------|-------------|
| A1 | Логошот_Умное облако | 1→2→3→5→7 | Полный логошот: структура, тайминг, shape paths логотипа, цвета, easing |
| A2 | Логошот без саблайна | 1→3 | Различия с A1 (без текста) |
| A3 | Логошот без саблайна с плашкой | 1→2→3 | Параметры плашки (shape, color, animation) |
| A4 | Логошоты вертикальный_Умное облако | 1→3 | Различия с A1 для 9:16 (позиции, размеры) |

**Ожидаемый результат:** JSON с полными keyframes логотипа, саблайна, плашки + vertices логотипа.

---

### Блок B: Текстовые блоки (проекты Titles + SMM_Pack)

**Открыть:** `Titles/Titles (3).aep`

| # | Композиция | Шаги | Что получаем |
|---|-----------|------|-------------|
| B1 | Подпись_спикеров | 1→2→3→4 | Lower third: структура слоёв, размеры текста, анимация появления/ухода, маски |

**Открыть:** `SMM_Pack/SMM_pack.aep`

| # | Композиция | Шаги | Что получаем |
|---|-----------|------|-------------|
| B2 | Плашки_1x1 | 1→2→3→4 | Shape-подложка + текст: цвета, размеры, keyframes, маски |
| B3 | Плашки_9x16 | 1→3 | Различия с B2 для 9:16 |

**Ожидаемый результат:** Параметры lower third и text card с shape-подложкой.

---

### Блок C: Transitions (проект SMM_Pack) — опционально для MVP

**Открыть:** `SMM_Pack/SMM_pack.aep`

| # | Композиция | Шаги | Что получаем |
|---|-----------|------|-------------|
| C1 | Transition_gray | 1→2→3 | Wipe-переход: shape layers, цвета, keyframes |
| C2 | Transition with Logo | 1→3→7 | Logo в середине transition |
